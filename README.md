# Geo Fusion Quiz

Single app with two playable scopes:

1. World Countries
2. China Provinces (PRC ADM1)

Each round has five skills:

1. Outline recognition
2. Map placement
3. Capital naming
4. Population estimate
5. Area estimate

## Stack

- React + TypeScript + Vite
- Local storage progress tracking
- Static JSON registries generated by Node scripts

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Build data bundles

Generate both registries:

```bash
npm run build:data
```

Or individually:

```bash
npm run build:data:countries
npm run build:data:prc
```

Output files:

- `public/data/country-registry.json`
- `public/data/prc-province-registry.json`

## Tests

Run 10 logic smoke tests:

```bash
npm run test:10
```

## Scoring rules

- Location: inside boundary = `100`; otherwise score decays by distance.
- Population and area (same tiers):
  - `e <= 10%` -> `100`
  - `e <= 25%` -> `75`
  - `e <= 40%` -> `50`
  - `e <= 50%` -> `30`
  - `e > 50%` -> `0`
  where `e = |guess - true| / true`
